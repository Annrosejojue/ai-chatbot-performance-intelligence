End‑to‑end analysis of an enterprise BI chatbot, including performance evaluation, confidence calibration, user satisfaction modeling, and business impact insights.
This project analyzes real organizational interactions with an AI-powered BI chatbot.  
It evaluates chatbot performance, confidence calibration, and user satisfaction.  
The dataset is cleaned, explored, and modeled to uncover business impact patterns.  
A performance scoring engine and analytics dashboard provide actionable insights.  
Built as a 5‑day end‑to‑end project using Python, Colab, and modern data techniques.

Models Trained- Linear Regression - Random Forest - Gradient Boosting - XGBoost - LightGBM - SVR - KNN 
Leaderboard Metrics Evaluated using: - MAE - RMSE
